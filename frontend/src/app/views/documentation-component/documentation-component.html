Quick Start Guide

1

Create an Account

Sign up using your email, Google, or GitHub account. Verification is instant with OAuth providers.

Terminal
# No installation required
# Visit https://codecollab.com
# Click "Sign Up" and choose your provider
2

Create Your First Project

Start a new project or import from GitHub. Choose from templates or start blank.

Workspace
1. Click "Create Project" in sidebar
2. Enter project name and description
3. Choose template (Optional)
4. Click "Create"
3

Invite Collaborators

Share your project with team members by sending email invitations with specific permissions.

Project Settings
1. Open project settings
2. Navigate to "Collaborators"
3. Enter email addresses
4. Set permissions (View/Edit)
5. Send invitations

Frequently Asked Questions

How many collaborators can join a session?

Currently, we support up to 10 concurrent collaborators per project session. For enterprise needs, contact us for higher limits.

Is my code secure?

Yes! All code is encrypted in transit (HTTPS) and at rest. We use JWT tokens for authentication and never store plain-text passwords.

Can I use CodeCollab offline?

While CodeCollab requires internet for real-time collaboration, you can export projects to work locally and sync when back online.
